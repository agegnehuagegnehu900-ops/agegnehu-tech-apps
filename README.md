# Agegnehu Tech - German Language Master

A React Native (Expo) mobile application for learning the German language with Amharic translations. Built by **Agegnehu Shibiru** under the **Agegnehu Tech** brand.

## Features

- **Chapter-based learning**: Organized lessons covering alphabet, greetings, numbers, grammar, daily conversations, and a dictionary
- **Bilingual content**: German vocabulary with Amharic (and English) translations
- **Offline-first**: Designed to work without internet connectivity
- **About the Developer**: Professional author page with biography in both Amharic and English

## Chapters

1. **Introduction & Alphabet** - German letters and special characters
2. **Greetings** - Formal and informal greetings
3. **Numbers (1-1000)** - Counting and number pronunciation
4. **Grammar: Articles** - Der, Die, Das and noun genders
5. **Daily Conversations** - Practical phrases for work and shopping
6. **A-Z Dictionary** - Common vocabulary reference

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Installation

```bash
npm install
```

### Running the App

```bash
# Start the development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios

# Run on Web
npm run web
```
